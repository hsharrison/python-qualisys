Load data from a .txt file exported from QTM into `pandas <http://pandas.pydata.org>`_::

    pip install qualisys

.. code-block:: python

    In [1]: from qualisys import load_qtm_data

    In [2]: data, metadata = load_qtm_data('example_data.txt')
